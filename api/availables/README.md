# Availables API

`GET` /availables/os<br>
Fetch list of available OS.

Response
```json
{
    "data": [
        "ios",
        "android",
        ""
    ]
}
```

`GET` /availables/size
Fetch list of available sizes.

Response
```json
{
    "data": [
        {
            "height": 320,
            "width": 240
        },
        {}
    ]
}
```
